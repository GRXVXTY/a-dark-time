# A Dark Time - Character Creation System

A gothic-themed character creation system for the "A Dark Time" roleplay server.

## Features

- Character Creation with detailed stat charts
- Monster Creation system
- Weapon Creation system
- Gadget Creation system
- Dark, gothic theme
- Local storage for saving creations
- Responsive design
